CSS选择器备忘录

常用选择器

兼容性: IE6 只注意 :active&:hover 忽略 :focus, IE7 支持 hover 忽略其他
常用的选择器

  • 类型选择器(元素选择器)
  • 后代选择器

尽量通过集合类型, 后代, ID 和类 来获取自己想要的内容;

伪类

  • 链接伪类
    • :link
    • :visited
  • 动态伪类()
    • :hover
    • :active
    • :focus

P.S 通过连接伪类可以创建更复杂的行为. a:visited:hover { color: olive; }

通用选择器

与其他选择器结合使用时, 通用选择器可以用来对某个元素的 所有后代 应用样式.

高级选择器

兼容性: IE6- 不支持, 因此在站点功能或布局很重要的任何元素上应尽量避免使用这些高级选择器.

子选择器和相邻同胞选择器

子选择器, 只选择元素的直接后代

以上的代码内容不会影响到列表中的列表内容;

兼容性: IE7+支持, 但 IE7 中若父元素和子元素之间有 HTML 注释就会出问题.

QS: 如何在 IE6 及更低版本中模拟子选择器的效果.

相邻同胞选择器, 定位同一个父元素下某个元素之后的元素.

兼容性: IE7 中如果在目标元素之间有注释会出现问题.

属性选择器

兼容性: IE6 不支持属性选择器

层叠和特殊性

层叠的重要度次序

  1. 标有 !important 的用户样式
  2. 标有 !important 的作者样式
  3. 作者样式
  4. 用户样式
  5. 浏览器/用户代理应用的样式

选择器特殊性的4个成分等级: a,b,c,d -> abcd (1,0,1,2 -> 1012);

  • 样式是行内样式 style="", 那么 a = 1
  • b 等于 id 选择器的总数
  • c 等于类, 伪类和属性选择的数量
  • d 等于类型选择器和伪元素选择器的数量

总结: 行内 > id > class, 伪类 > 类型选择器,伪元素选择器
